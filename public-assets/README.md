# Public Assets

This directory contains images and assets used in the SuperGitSight documentation and README.

## Required Images

The following images should be copied from `github-analyzer/public/` and `github-analyzer/public/app-about/`:

- `supergitsight.png` - SuperGitSight logo (from `public/supergitsight.png`)
- `search-page.png` - Search page screenshot (from `public/app-about/search-page.png`)
- `user-profile-analyze-hero.png` - Profile analysis dashboard (from `public/app-about/user-profile-analyze-hero.png`)
- `profile-comparison.png` - Profile comparison view (from `public/app-about/profile-comparison.png`)
- `kcrismag-framework.png` - KCRISMAG framework breakdown (from `public/app-about/kcrismag-framework.png`)
- `acid-framework.png` - ACID framework overview (from `public/app-about/acid-framework.png`)
- `languages-breakdown.png` - Language breakdown visualization (from `public/app-about/languages-breakdown.png`)

## Image Guidelines

- **Format**: PNG or SVG preferred
- **Size**: Optimize for web (keep file sizes reasonable)
- **Dimensions**: 
  - Logo: 96x96px or larger (square)
  - Screenshots: 800px width recommended for README
- **Quality**: High quality but optimized for web

## Placeholder Images

If you don't have actual screenshots yet, you can:
1. Use placeholder images from services like [Placeholder.com](https://placeholder.com)
2. Create simple mockups
3. Use screenshots from the live application

## Updating Images

When updating images:
1. Replace the existing file with the same filename
2. Ensure the file size is optimized
3. Update any references in documentation if dimensions change significantly
